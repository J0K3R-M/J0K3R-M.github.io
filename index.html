portfolio

As a cybersecurity student, I have worked on various projects that demonstrate my skills in ethical hacking, network security, and secure application development. Below are some highlights of my work:

Penetration Testing

I performed comprehensive penetration testing on a web application to identify and exploit security flaws. Utilizing tools like Burp Suite and OWASP ZAP, I uncovered vulnerabilities such as SQL injection, cross-site scripting (XSS), and broken authentication. I provided detailed reports and recommendations to help the development team enhance the security of their application.

Secure Coding Practices

I developed a secure login system using Python and Flask, implementing advanced encryption techniques like bcrypt for password hashing and secure session management. The system includes multi-factor authentication (MFA) and rate limiting to protect against brute force attacks. This project showcases my ability to write secure and reliable code.

Network Security

I designed and implemented a secure network architecture for a small business, including setting up firewalls, VPNs, and intrusion detection systems. I conducted regular security audits and provided training to employees on best practices for maintaining network security. This project highlights my expertise in network security and risk management.
